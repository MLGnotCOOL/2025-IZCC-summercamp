暑訓簡介

帶你認識IZCC四校聯合暑訓

由建中資訊（INFOR）、中山資研（ZSISC）、成功電研（CKCSC）、景美電資（CMIOC）舉辦一年一度的大型活動！參加四校聯合暑訓會是你上高中前最不後悔的決定！

在活動的七天中，你可以：

  • 認識更多升高中的朋友，拓展你的交友圈
  • 更熟悉高中生活，認識更多學長姊
  • 學習更多關於Python、C++、tkinter、網頁設計的知識
  • 參觀資訊產業公司
  • 遊玩各種有趣的神祕活動！

活動介紹

在七天中的六天，我們都有安排有趣的活動

第一天-大地遊戲

大地遊戲又名曠野遊戲、越野遊戲。在這次的暑訓中，我們會於中正紀念堂由玩大地遊戲， 過程中會有許多好玩、刺激的關卡，每一小隊會在大地遊戲中盡全力在每一關中拿取分數，並且在最後爭取最高名次！

第二天-企業參訪

以後想成為工程師嗎？那你絕對不能錯過這項活動！在這一天，我們會進行科技業公司的參訪，至於要參訪的公司呢，就請你拭目以待啦！

第三天-RPG

只有玩過線上版的 RPG 嗎?那你一定要來體驗看真人版 RPG！有著燒腦的故事劇情，以及刺激的遊戲關卡等你來破解。想看到完整的劇情內容的話，一定要來參加！

第四天-團康活動

在日常生活中，我們常常因為生活過於忙碌而忘了關心周遭。而在這個活動中，我們將用一整天的時間，帶領你們更加認識雙北，敬請期待！ 至於詳細活動內容，等你來就知道了！

第五天-趣味遊戲

這個活動將搭配許多桌遊及神秘小遊戲。你需要做的，是贏下一次次的奮鬥、一場場的交戰，並在最後讓自己的小隊成為最終的贏家！

第七天-神秘活動

為此營隊中最精彩的活動，會讓你們有個難忘的夜晚，如果你們知道內容的話，這樣就不好玩啦！敬請期待！

課程介紹

除了活動以外，我們也設計了許多不同的資訊相關課程，一起來認識這些課程吧！
我們將以淺顯易懂的方式教學，就算是零基礎也能完全學會喔！

Python與曼陀羅繪圖

Python 是一種簡單易學的程式語言，適合入門。從 Python 的基礎語法開始，再利用 turtle 模組、數學與邏輯畫出對稱又吸睛的曼陀羅。 邊學程式邊創作，體驗科技與美感的結合！


在課程中包含：

  • 認識Python的基礎語法
    • 輸入輸出
    • 運算子
    • for迴圈
  • 使用 Turtle 模組繪圖
  • 結合數學公式與邏輯
  • 創作幾何圖案與曼陀羅
PyQt

PyQt 是一個可以建立圖形使用者介面（GUI）應用程式的 Python 模組，可以製作各種精美的工具、小遊戲等。 藉由 PyQt，你可以輕鬆的做出視覺化應用程式，也可以搭配 Qt Designer 更直觀的做出更加美觀、更加現代化的介面 ~

在這門課程中，你可以學到：

  • 建立圖形化應用程式
  • 設計各種經典 GUI 元件
  • 視覺化應用程式介面設計技巧
  • 處理元件操作的程式設計邏輯
  • 如何使用 Qt Designer 設計應用程式
    • 網頁設計

    INFOR 38th

    建國中學資訊社38屆

    建國中學資訊社是一個以電腦科技為主題的社團，歡迎對程式設計、網頁製作、遊戲開發、人工智慧等領域有興趣的同學加入。 在資訊社，你可以學習到最新的資訊技術，並且與其他社員一起分享作品與想法。

    CKCSC 38th

    成功高中電子計算機研習社38屆

    「成功高中電子計算機研習社 Cheng Gong Computer Study Club」簡稱成功電研CKCSC，是一個成功高中內的學術性社團，在社課會教一些關於資訊領域的課程， 會有學長帶你從 0開始學，也會有進階班，歡迎學弟加入電研社的大家庭！ 除了教學課程，我們也與IZCC分別是建中資訊、中山資研、景美電資等三校，一同合辦許多活動與聯課， 並且也會在放學後與四校共同開設課程，提供四校學弟妹與友校們交流的機會。

    ZSISC 32nd

    中山女高資訊研究社32屆

    中山資研，又稱楓資，全名為中山女高資訊研究社，社課主要以教學 C++與網頁設計為主，無論是程式新手或是電神，都可以跟幹部們一起學習，我們也會分享有關於資訊方面的事情， 在放學後也有自由選擇是否參加的不同課程，同時我們也有許多與友校交流的活動，只要對於資訊方面有興趣的學妹，或是想嘗試看看的，都歡迎加入楓資！

    CMIOC 32nd

    景美女中電腦資訊社32屆

    歡迎來到景美女中的電腦資訊社！我們是一個致力於推廣科技與計算機知識的社團，擁有豐富的學習與實踐機會。 不論你是對編程充滿熱情的新手，還是想進一步探索電腦科學領域的高手，我們都歡迎你加入我們的行列。
